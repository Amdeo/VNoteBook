# 流程控制

```
if  condition
then
    statement(s)
fi
```

```
if  condition;  then
    statement(s)
fi
```

```
if  condition
then
   statement1
else
   statement2
fi
```

```
if  condition1
then
   statement1
elif condition2
then
    statement2
elif condition3
then
    statement3
……
else
   statementn
fi
```
