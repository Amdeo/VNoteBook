# 默认参数

`默认参数可以有个`
`默认参数必须放在后面`

```C++
#include <iostream>

using namespace std;

void fun(int a,int b = 1,int c = 1)
{
    cout << "一个默认参数" <<endl;
    cout << "a: " << a << endl;
    cout << "b: " << b << endl;
    cout << "c: " << c << endl;
}


int main(int argc, char const *argv[])
{
    fun(1);
    return 0;
}

```
