# 指针
## 定义一个指针

```
datatype *name;
```
```
int a = 100;
int * p;
p = &a;    //指针变量定义只后再赋值

int a = 199;
int *p1 = &a; //指针的初始化
```

通过指针获取变量的值
```
int a = 15;
int *p = &a;

printf("%d, %d\n", a, *p);  //两种方式都可以输出a的值
```

指针变量进行表达式运算
```
int x, y, *px = &x, *py = &y;
y = *px + 5;  //表示把x的内容加5并赋给y，*px+5相当于(*px)+5
y = ++*px;  //px的内容加上1之后赋给y，++*px相当于++(*px)
y = *px++;  //相当于y=(*px)++
py = px;  //把一个指针的值赋给另一个指针
```
